# image-sender

Email images

### How to run?

- `npm install`
- `npm start`

### How to deploy?

- `npm run build:prod`
- `npm run deploy:prod`

### Requirements

- NodeJS >= 16

### Tech stack

- ReactJS
- Sass
- Webpack
- Babel
